This is the homepage for the nsm site repository template basic-site [BitBucket, GitHub, GitLab]. Nift (nsm) is a cross-platform open source git-like and $\LaTeX$-like site manager coded in C++.

You can import (or fork and rename) the basic-site site repository template to begin making your own site with the same design.

See the About page for information about adding/removing pages to/from your site and modifying the content/layout of your site.

See the FAQ page for how to import/fork this repository along with answers to other frequently asked questions.
